# Notes Keeper App

A simple and responsive Notes Keeper Application built using HTML, CSS, and JavaScript.
This app helps users create, save, edit, and delete notes easily.

## 🚀 Features

* Add new notes
* Edit existing notes
* Delete notes
* Responsive design
* Simple and clean UI
* Fast and lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run the Project

1. Download or clone the repository

```bash
git clone https://github.com/Bhoomika07jais/Notes-keeper-app.git
```

2. Open the project folder

3. Run `index.html` in your browser
